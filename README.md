# EmulationSystemFrontend

基于 [Element Plus](https://element-plus.org/) 的电路物联网仿真前端，具体请见：[https://modao.cc/app/6XdQIh9Es69j5wU7F934Ol](https://modao.cc/app/6XdQIh9Es69j5wU7F934Ol)

## 项目配置

```bash
pnpm install
pnpm run dev    # 开发
pnpm run build  # 部署
```

生成的部署文件位于 `./dist/`。
